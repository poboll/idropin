## 快速开始

### 注册账户

1. 访问 https://idrop.in
2. 点击右上角的"注册"按钮
3. 填写注册信息：
   - 用户名：3-20个字符
   - 邮箱：有效的邮箱地址
   - 密码：6-20个字符，建议包含字母、数字和特殊字符
   - 确认密码：再次输入密码
4. 点击"注册"按钮完成注册

### 登录系统

1. 访问 https://idrop.in
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录需要验证邮箱（如已启用）

### 忘记密码

1. 在登录页面点击"忘记密码"链接
2. 输入注册时的邮箱地址
3. 查收邮箱中的重置密码链接
4. 点击链接设置新密码
5. 使用新密码登录系统

## 用户账户

### 个人信息管理

#### 查看个人信息

1. 登录后点击右上角头像
2. 选择"个人设置"
3. 可以查看和编辑：
   - 用户名
   - 邮箱
   - 注册时间
   - 最后登录时间

#### 修改密码

1. 进入"个人设置"页面
2. 点击"修改密码"按钮
3. 输入当前密码
4. 输入新密码
5. 确认新密码
6. 点击"提交"按钮

**密码安全建议**：
- 至少8个字符
- 包含大小写字母、数字和特殊字符
- 不要使用个人信息（生日、手机号）
- 定期更换密码

#### 头像设置

1. 进入"个人设置"页面
2. 点击"上传头像"按钮
3. 选择图片（支持JPG、PNG、GIF格式，最大2MB）
4. 裁剪图片（系统提供裁剪工具）
5. 点击"保存"按钮

## 文件管理

### 上传文件

#### 单文件上传

1. 登录后进入"文件管理"页面
2. 点击"上传文件"按钮
3. 选择要上传的文件
4. 系统自动开始上传并显示进度条
5. 上传完成后显示文件信息

**支持的文件类型**：
- 文档：PDF、Word、Excel、PPT、TXT
- 图片：JPG、PNG、GIF、BMP
- 视频：MP4、AVI、MKV
- 音频：MP3、WAV、AAC
- 压缩包：ZIP、RAR、7Z

**文件大小限制**：
- 免费用户：100MB
- 付费用户：500MB

#### 多文件上传

1. 点击"上传文件"按钮
2. 选择多个文件（最多10个）
3. 点击"开始上传"按钮
4. 系统依次上传文件并显示每个文件的上传进度

#### 大文件上传

对于超过100MB的文件，系统自动启用分片上传：

1. 选择大文件上传
2. 系统自动将文件分割为5MB的分片
3. 依次上传每个分片
4. 支持断点续传（网络中断后可继续上传）
5. 所有分片上传完成后自动合并

**优势**：
- 网络中断不影响上传进度
- 支持暂停和继续上传
- 上传失败可重试失败的分片

### 文件管理

#### 文件列表

文件列表支持以下功能：

1. **排序**：
   - 按名称排序（升序/降序）
   - 按大小排序（升序/降序）
   - 按上传时间排序（升序/降序）

2. **筛选**：
   - 按文件类型筛选
   - 按分类筛选
   - 按标签筛选

3. **搜索**：
   - 支持文件名搜索
   - 支持文件内容搜索

4. **分页**：
   - 每页显示20个文件
   - 支持页码跳转

#### 文件操作

1. **下载文件**：
   - 点击文件右侧的下载按钮
   - 支持断点续传
   - 浏览器自动保存到下载文件夹

2. **预览文件**：
   - 点击文件右侧的预览按钮
   - 支持在线预览图片、PDF、视频、音频
   - 无需下载即可查看内容

3. **重命名文件**：
   - 右键点击文件
   - 选择"重命名"
   - 输入新文件名
   - 按回车确认

4. **移动文件**：
   - 选择一个或多个文件
   - 拖拽到目标文件夹
   - 或右键选择"移动到..."

5. **复制文件**：
   - 右键点击文件
   - 选择"复制"
   - 系统创建文件副本

6. **删除文件**：
   - 选择一个或多个文件
   - 点击删除按钮
   - 系统提示确认删除
   - 删除的文件移至回收站（保留30天）

7. **批量操作**：
   - 支持批量下载（打包为ZIP）
   - 支持批量移动
   - 支持批量删除

### 文件分类

#### 创建分类

1. 点击左侧"分类"菜单
2. 点击"新建分类"按钮
3. 输入分类名称（1-50字符）
4. 选择父分类（可选）
5. 输入分类描述（可选）
6. 点击"保存"按钮

**分类类型**：
- 文档：PDF、Word、Excel、PPT
- 图片：JPG、PNG、GIF
- 视频：MP4、AVI、MKV
- 音频：MP3、WAV
- 其他：其他文件类型

#### 管理分类

1. 查看分类树形结构
2. 右键点击分类可以：
   - 编辑分类信息
   - 删除分类
   - 移动分类（拖拽）
3. 将文件拖拽到分类即可归类

#### 文件标签

1. 选择文件
2. 右键点击"添加标签"
3. 输入标签名称
4. 按回车添加
5. 一个文件可以添加多个标签

**标签使用**：
- 快速筛选文件
- 创建自定义标签组
- 标签颜色区分

## 收集任务

### 创建任务

1. 点击"收集任务"菜单
2. 点击"新建任务"按钮
3. 填写任务信息：
   - 任务标题：1-100字符，必填
   - 任务描述：0-500字符，可选
   - 截止日期：选择日期和时间
   - 允许的文件类型：选择文件类型
   - 最大文件大小：设置文件大小限制（MB）
   - 最大提交次数：设置提交次数限制
   - 是否需要登录：勾选是否需要登录才能提交

4. 点击"创建"按钮

**任务状态**：
- 进行中（ACTIVE）：任务开放，可以提交文件
- 已关闭（CLOSED）：任务已结束，不能再提交文件

### 提交文件

#### 登录用户提交

1. 进入任务详情页面
2. 点击"提交文件"按钮
3. 选择要提交的文件（最多10个）
4. 点击"确认提交"按钮
5. 系统显示提交成功提示

#### 匿名提交

1. 进入任务详情页面
2. 输入提交者姓名（必填）
3. 输入提交者邮箱（必填）
4. 选择要提交的文件
5. 点击"确认提交"按钮
6. 系统生成提交编号供查询

**提交限制**：
- 单次最多提交10个文件
- 每个文件最大100MB
- 同一邮箱最多提交20次

### 查看提交

1. 进入任务详情页面
2. 查看所有提交记录
3. 提交记录包含：
   - 提交者信息
   - 提交时间
   - 提交的文件列表
   - 提交状态

### 任务统计

任务详情页面显示统计信息：

1. **总提交数**：所有提交的总数
2. **唯一提交者**：不同提交者的数量
3. **文件类型分布**：按文件类型的统计（饼图）
4. **提交时间趋势**：最近7天的提交趋势（折线图）

### 任务管理

1. **编辑任务**：
   - 修改任务标题、描述
   - 调整截止日期
   - 更新文件类型和大小限制
   - 关闭/重新开启任务

2. **删除任务**：
   - 删除任务及其所有提交记录
   - 删除前需要确认

3. **分享任务**：
   - 生成任务分享链接
   - 支持设置访问密码
   - 支持设置有效期

## 文件分享

### 创建分享

1. 选择要分享的文件
2. 点击"分享"按钮
3. 设置分享选项：
   - 访问密码：4-20字符（可选）
   - 有效期：1-30天（可选）
   - 最大下载次数：1-100次（可选）
4. 点击"生成分享链接"按钮

**分享链接格式**：
- https://idrop.in/s/ABC12345
- 8位随机分享码

### 访问分享

#### 无密码分享

1. 访问分享链接
2. 系统直接显示文件信息
3. 可以下载或预览文件

#### 有密码分享

1. 访问分享链接
2. 系统提示输入访问密码
3. 输入正确密码后显示文件信息
4. 输入错误密码5次后链接失效

**密码安全建议**：
- 不要使用简单密码（123456、password等）
- 使用字母、数字和特殊字符组合
- 定期更换密码

### 分享管理

1. 点击"我的分享"菜单
2. 查看所有创建的分享
3. 分享信息包括：
   - 分享链接
   - 创建时间
   - 访问次数
   - 最大下载次数
   - 有效期
   - 是否有密码

4. 可以进行以下操作：
   - 取消分享（删除分享链接）
   - 复制分享链接
   - 查看访问记录

## 常见问题

### 账户相关

**Q: 忘记密码怎么办？**

A: 
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收重置密码邮件
4. 点击邮件中的链接设置新密码

**Q: 无法登录怎么办？**

A:
1. 检查用户名和密码是否正确
2. 清除浏览器缓存和Cookie
3. 尝试使用其他浏览器
4. 检查网络连接
5. 联系客服支持

**Q: 如何修改邮箱？**

A:
1. 登录后进入"个人设置"
2. 点击"修改邮箱"
3. 输入新邮箱
4. 验证新邮箱（发送验证码）
5. 确认修改

### 文件相关

**Q: 上传失败怎么办？**

A:
1. 检查文件大小是否超过限制（免费用户100MB，付费用户500MB）
2. 检查文件类型是否支持
3. 检查网络连接是否稳定
4. 尝试使用分片上传（大文件自动启用）
5. 刷新页面重试

**Q: 下载失败怎么办？**

A:
1. 检查网络连接
2. 清除浏览器下载历史
3. 尝试使用其他浏览器
4. 检查磁盘空间是否充足
5. 联系客服支持

**Q: 文件预览失败？**

A:
1. 检查文件类型是否支持预览
2. 更新浏览器到最新版本
3. 尝试下载后查看
4. 检查是否安装了必要的插件

**Q: 如何批量下载文件？**

A:
1. 选择多个文件（最多10个）
2. 右键点击"批量下载"
3. 系统自动打包为ZIP文件
4. 下载ZIP文件
5. 解压ZIP文件即可

### 存储相关

**Q: 存储空间不足怎么办？**

A:
1. 删除不需要的文件
2. 清空回收站
3. 升级到付费计划获得更多空间
4. 联系客服申请扩容

**Q: 如何查看存储使用情况？**

A:
1. 登录后进入"数据分析"
2. 查看"存储使用"部分
3. 显示已用空间和剩余空间
4. 显示各类型文件占用空间

### 分享相关

**Q: 分享链接失效怎么办？**

A:
1. 检查是否超过有效期
2. 联系分享者重新生成分享链接
3. 检查分享密码是否正确
4. 最多尝试5次，超过后需要重新生成

**Q: 如何取消分享？**

A:
1. 进入"我的分享"页面
2. 找到要取消的分享
3. 点击"取消分享"按钮
4. 确认取消

**Q: 分享链接可以设置多长时间？**

A:
1. 最短1天
2. 最长30天
3. 到期后自动失效
4. 到期前可以续期

### 移动端相关

**Q: 如何安装到手机？**

A:
1. 使用Chrome、Edge、Safari等现代浏览器访问 https://idrop.in
2. 点击地址栏的安装图标
3. 确认添加到主屏幕
4. 完成安装

**Q: 离线时能使用哪些功能？**

A:
1. 浏览已缓存的文件
2. 查看文件列表
3. 查看个人设置
4. 查看任务列表

**Q: 如何节省流量？**

A:
1. 在设置中开启"省流量模式"
2. 仅在WiFi下加载高清图片
3. 移动网络下使用低质量图片
4. 关闭自动播放视频
